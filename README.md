<br />

## Introduce

> 외출할 때마다 어떤 옷을 입을지 고민을 합니다.  
> 고민이 길어질 수록, 낭비되는 시간과 더러워지는 방은 하루의 기분을 망치기도 합니다.
>
> `오늘의 옷`은 여러분의 스타일을 아카이빙하고 결정을 돕는 서비스입니다.
>
> 옷을 여러 번 갈아입을 필요 없이, 스타일 목록에서 오늘 입을 옷을 선택할 수 있습니다.  
> 또한 이용자의 특징 및 선호를 바탕으로 검색 및 추천 서비스를 제공합니다.
